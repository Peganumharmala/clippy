# Clippy

New feature that allows you to link cloud LLM providers directly to Clippy.

How can you do it?
1. Open Settings from the main menu.
2. Select "Cloud API".
3. Choose your preferred AI Provider (OpenAI, Claude, Gemini, etc).
4. Enter the Model ID (if you're unsure, just ask your AI for its current identifier).
5. Paste your API Key.
6. Click Save, and you're all set!
